# results/

Generated outputs — written *from code*, never hand-edited.

- `tables/` — regression/summary tables as `.tex`, `\input{}` into the paper and slides.
- `figures/` — figures as `.pdf` (vector, for LaTeX) + `.png`.
- `diagnostics/` — EDA / diagnostic plots.

Numbers in the paper trace back to the cell that produced them (the `verifier` agent checks this).
